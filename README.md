# my-skills

Personal agent skills created by Abner Soares Aves Junior.

## Installation

Install all skills using `npx skills add`:

```bash
npx skills add abnersajr/my-skills
```

Or install specific skills:

```bash
npx skills add abnersajr/my-skills --skill beads-github-sync
```

## Skills

- `beads-github-sync` - Sync Beads issues with GitHub Issues via beads-synced GitHub Action
- `beads-task-breakdown` - Break features into hierarchical Beads issues with dependencies
- `beads-workflow` - Manage Beads issue tracker lifecycle (ready → claim → work → update → close)
- `beads-worktree` - Work with Beads in git worktree setups using BEADS_DIR
- `model-routing` - Route subagent tasks to appropriate Xiaomi MiMo model

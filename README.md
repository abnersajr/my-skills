# my-skills

Personal agent skills created by Abner Soares Aves Junior.

## Installation

Install all skills using `npx skills add`:

```bash
npx skills add abnersajr/my-skills
```

Or install specific skills:

```bash
npx skills add abnersajr/my-skills --skill beads-github-sync
```

## Skills

- `beads-github-sync` - Sync Beads issues with GitHub Issues
- `beads-task-breakdown` - Break features into Beads issues
- `beads-workflow` - Work with Beads issue tracker (bd CLI)
- `beads-worktree` - Work with Beads in git worktree setups
- `model-routing` - Determines which Xiaomi MiMo model to use for subagents
